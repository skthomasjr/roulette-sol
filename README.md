### Blockchain-based Mathematically Fair Roulette Table

roulette-sol is a mathematically fair roulette table that exists as multiple cooperative smart contracts on the Ethereum VM. Work is in progress!!!

```
git
docker
docker-compose
```

```
npm run test
rpm run workspace
npm run cleanup
```
